M8 - First Sale


Introducing the MVP to customers

At this point, the implementation of the web application did not contain a lot of functionalities, as seen in the MVP, so the expectation for the first sale were not so high. However, we decided to have a first overview of what the possible customers think about the current state.

Given the fact that the time was quite limited, we called for help for this milestone to a couple of friends which might be interested in organizing some private events. The website is an early prototype and further functionalities wil be added in a later release.

The feedback that we got from some of the users that accessed our website was expected, being that it is still unfinished and there are many features which are not developed yet.

User Feedback

  • The site is pretty easy to use, the user interface is friendly, but it does have some problems regarding to its functionality. When adding an event, you can't choose the time interval in which your event is taking place or slots for multiple artists. Also, the Details page for the artists does not give enough information.
  • The site currently does not have any more features than the existing platforms, but it seems nice that it is a site created solely for artists and event organisers. The basic functionality is here, but i would like to see more features in the future, like login based on existing accounts from other social media or google maps integration for the location of the events.
